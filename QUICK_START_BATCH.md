# دليل البدء السريع - إضافة الموظفين بالدفعة

## الخطوة 1: تثبيت المكتبة المطلوبة
```bash
composer require phpoffice/phpspreadsheet
```

## الخطوة 2: مسح Cache
```bash
php artisan config:clear
php artisan cache:clear
```

## الخطوة 3: استخدام الميزة

### 1. اذهب إلى صفحة المستخدمين
- اضغط على "إدارة المستخدمين" في القائمة

### 2. اضغط على زر "إضافة الموظفين بالدفعة"
- الزر الأخضر بجانب "إضافة مستخدم جديد"

### 3. حمّل قالب Excel
- اضغط على "تحميل قالب Excel"
- املأ البيانات في الملف

### 4. ارفع الملف
- اختر ملف Excel المملوء
- اختر القسم والمنصب الافتراضي
- قم بربط الأعمدة مع الحقول
- اضغط "إضافة الموظفين"

## الحقول المطلوبة فقط
- **الاسم** (Name)
- **البريد الإلكتروني** (Email)

## باقي الحقول اختيارية
يمكنك ربط أي عمود في ملف Excel مع أي حقل في النظام.

## ملاحظات مهمة
- الملف يجب أن يكون بصيغة .xlsx أو .xls
- الحد الأقصى للحجم: 10 ميجابايت
- يجب أن يحتوي على صف رؤوس
- يمكنك معاينة البيانات قبل الرفع
