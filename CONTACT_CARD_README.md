# نظام بطاقات الاتصال الشاملة

## نظرة عامة
تم تطوير نظام بطاقات الاتصال الشاملة لتوفير واجهة موحدة ومتكاملة لعرض وإدارة معلومات الاتصال الخاصة بالموظفين في النظام.

## المميزات الرئيسية

### 1. معلومات الاتصال الشاملة
- **أرقام الهواتف**: العمل، المنزل، الشخصي، المحمول، الطوارئ
- **وسائل التواصل الاجتماعي**: واتساب، تيليجرام، سكايب، فيسبوك، إنستجرام، لينكد إن
- **معلومات العمل**: التحويلة، المكتب، موقع العمل

### 2. معلومات العمل التفصيلية
- رقم الموظف
- تاريخ التعيين
- المسمى الوظيفي
- موقع العمل
- المكتب والتحويلة
- المدير المباشر والمرؤوسون

### 3. المعلومات الشخصية
- تاريخ الميلاد
- الجنسية
- العنوان الكامل
- المهارات والاهتمامات
- اللغات المتقنة
- نبذة شخصية

### 4. إعدادات الخصوصية
- إظهار/إخفاء أرقام الهواتف
- إظهار/إخفاء الإيميل
- إظهار/إخفاء العنوان
- إظهار/إخفاء وسائل التواصل الاجتماعي

### 5. وظائف التواصل السريع
- **رسالة سريعة**: إرسال رسائل عبر الإيميل، واتساب، أو تيليجرام
- **جدولة اجتماع**: جدولة اجتماعات مع الموظف
- **مكالمة سريعة**: الاتصال مباشرة بالأرقام المتاحة

### 6. النشاط الأخير
- عرض المهام الأخيرة المكلف بها
- عرض الطلبات الأخيرة
- تاريخ التفاعلات

### 7. زملاء العمل
- عرض زملاء العمل في نفس القسم
- روابط سريعة لبطاقات الاتصال الخاصة بهم

## كيفية الاستخدام

### 1. عرض بطاقة الاتصال
```
GET /users/{user}/contact-card
```

### 2. تعديل بطاقة الاتصال
```
GET /users/{user}/edit-contact-card
PUT /users/{user}/update-contact-card
```

### 3. إرسال رسالة سريعة
```
POST /users/{user}/quick-message
```

### 4. جدولة اجتماع
```
POST /users/{user}/schedule-meeting
```

### 5. تحديث إعدادات الخصوصية
```
POST /users/{user}/privacy-settings
```

## الحقول الجديدة في قاعدة البيانات

### معلومات الاتصال
- `phone_mobile` - الهاتف المحمول
- `phone_emergency` - هاتف الطوارئ
- `whatsapp` - واتساب
- `telegram` - تيليجرام
- `skype` - سكايب
- `facebook` - فيسبوك
- `instagram` - إنستجرام

### معلومات العمل
- `employee_id` - رقم الموظف
- `hire_date` - تاريخ التعيين
- `work_location` - موقع العمل
- `office_room` - المكتب
- `extension` - التحويلة

### معلومات شخصية
- `birth_date` - تاريخ الميلاد
- `nationality` - الجنسية
- `address` - العنوان
- `city` - المدينة
- `country` - البلد

### معلومات إضافية
- `skills` - المهارات (JSON)
