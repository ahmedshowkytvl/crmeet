معلومات CSS المطبقة

  • direction: ltr - لضبط اتجاه input-group
  • text-align: right - لمحاذاة النص لليمين
  • transform: scaleX(-1) - لعكس الأيقونات
  • border-radius - لضبط الزوايا المدورة
  • border - لضبط الحدود بين العناصر
